# Module atlas

## Responsibility

Generated go-bindata package that embeds Hive operator configuration manifests (YAML) from `config/` directories -- including controllers, hiveadmission, RBAC, and sharded controllers.

## Public Interface/API

Generated by go-bindata. Key functions:

- `Asset(name string) ([]byte, error)` -- load embedded asset by name
- `MustAsset(name string) []byte` -- load asset or panic
- `AssetDir(name string) ([]string, error)` -- list files under an embedded directory
- `AssetInfo(name string) (os.FileInfo, error)` -- get file info for an embedded asset
- `AssetNames() []string` -- list all embedded asset names
- `RestoreAsset(dir, name string) error` -- write an embedded asset to disk
- `RestoreAssets(dir, name string) error` -- recursively restore embedded assets

## Internal Dependencies

- Standard library only (`fmt`, `os`, `path/filepath`, `strings`, `time`, `io/ioutil`)

## Capabilities

- Embeds YAML manifests for: controllers deployment, hiveadmission (webhooks, deployment, service, RBAC), sharded controllers (service, statefulset), RBAC roles/bindings
- Provides runtime access to operator deployment manifests without filesystem reads
